How to preview stuff in Codespaces?

Well, that's easy. Just start a server like browser-sync. The environment will notice that the server tries to open a port, creates a port forwarding and instructs the browser to load content. Note that this is technially a proxy setup, not a port forwarding. Not clear whether it works for anything but HTTP and HTTPS.

Let's find out by tring to run netcat. But: This is (like dig) not installed by default.

Installing stuff with apt on code spaces works like this: 

      sudo apt update # This is essential
      sudo apt install netcat

Forwarding a plain TCP port does not seem to work, At least I can not make it work.
